# React-CRUD


<h3>A Solo project built by me within 3 hours.</h3>
<br>

## Features :

 <p>--> Create product</p>
 <p>--> Edit & Update product</p>
 <p>--> Delete product</p>
 <p>--> View product</p>
 <p> --> Login </p>



## Tech Stack

- React.js
- Redux
- Thunk
- JSON Server
- axios


Start the server
```bash
npm run server
```
